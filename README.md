# Hero Avatar Interactive
![Hero Avatar Demo](assets/hero-avatar-demo.gif)

Hero Avatar Interactive is a p5.js project that showcases a culturally inspired hero avatar with interactive facial animations. Users can click the canvas to toggle the character's eyes open and closed while viewing an inspirational quote from Nelson Mandela.

This project demonstrates creative character design, basic animation, and user interaction using JavaScript and the p5.js library.

Features
🎨 Custom hero avatar created entirely with p5.js shapes.
👀 Click to open and close the avatar's eyes.
😊 Mouth color changes based on the avatar's expression.
🪶 Detailed accessories including a feather, beaded headband, earrings, and jewelry.
💬 Displays an inspirational Nelson Mandela quote.
🖱️ Simple mouse interaction for an engaging experience.
Technologies Used
JavaScript
p5.js
How to Run
Clone this repository.
Open the project folder.
Open the project using the p5.js Web Editor or run it with a local web server.
Click anywhere on the canvas to interact with the avatar.
Controls
Action	Result
Mouse Click	Toggle the avatar's eyes open and closed
